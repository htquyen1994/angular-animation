State Change

Angular cung cấp 3 khái niệm để định nghĩa state để sử dụng Animation Angular

  • Wildcard: Kiểu mặc định mọi state, chuyển state này đến bất kỳ state khác(Như router)
  • Void: Hiểu là khi khai báo state khi element đính vào DOM hoặc remove khỏi DOM
  • Custom: State do mình định nghĩa và chỉ định
