Overview

Khái niệm

Animation provides the illusion of motion, Animations can improve your application and user experience in a number of ways:

  • Without animations, web page transitions can seem abrupt and jarring
  • Motion greatly enhances the user experience, so animations give users a chance to detect the application's response to their actions
  • Good animations intuitively call the user's attention to where it is needed
Làm sao sử dụng Animation Angular
  • Evaluate data binding expression: Trạng thái nào thì animation sẽ được áp dụng cho element hay host
  • Target element, Host -> Nói cho angular biết phần tử nào được áp dụng
  • State: Nói cho angular biết là kiểu style CSS animation sẽ được áp dụng
  • Transition: Nói cho Angular biết quá trình chuyển đổi State sẽ như thế nào
    Một số khai function sẽ gặp khi làm việc về Animation Angular:
  • trigger()
  • style()
  • state()
  • animate()
  • transition()
  • keyframes()
  • group()
**Note***
  • Code style CSS: Camel Style or Dash Style
  • Lifecycle of Angular: After ngOnInit, Before ngOnDestroy
