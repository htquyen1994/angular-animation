Overview

1. Khái niệm

Animation được hiểu là hiệu ứng chuyển động, sử dụng để tạo hiệu ứng di chuyển cho các phần tử, giúp trải nghiệm người dùng được tốt hơn và được ứng dụng phổ biến trong các website hiện nay:

  • Nếu không có animation, Các phần tử trong trang web của bạn chuyển động một cách đột ngột và kh
  • Nâng cao trải nghiệm người dùng
  • Thu hút sự chú ý người dùng một cách trực quan đến nơi cần thiết

Vậy nhược điểm của nó mang lại là gì? Chính là performance của website bạn, tăng đáng kể. Nếu website bạn lớn và dùng nhiều animation thì performance sẽ là vấn đề của ban...

2. Animation Angular
  • Là một module trong angular (@angular/animations) Được xây dựng trên những tín năng của CSS
  • Lifecycle of Angular Animation: After ngOnInit, Before ngOnDestroy
  • Để sử dụng Angular Animation chúng ta cần import: BrowserAnimationsModule
  • Tương tự để tắt (disabled global) Angular Animation chúng ta import: NoopAnimationsModule

Evaluate data binding expression, Data binding target, State, Transition

Một số function sẽ gặp khi làm việc về Animation Angular
  • trigger()
  • style()
  • state()
  • animate()
  • transition()
  • keyframes()
  • group()
3. Angular animation so với CSS animation
  • Tạo animation nhanh hơn, dễ quản lý hơn so...(chạy nhanh hơn pure CSS animation?)
  • Support state trang thái :enter :leave cho DOM added hoặc removed
  • Automatic property calculation '*'
  • Animation callbacks: animation.start() và animation.done()
  • Tái sử dụng (Truyền parameter)
